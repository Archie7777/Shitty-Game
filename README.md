## 功能介绍：

#### 1.点击 “开始游戏” 开始游戏

#### 2.开始游戏后，点击右箭头按钮

#### 3.点击 “困难模式”，生成简单或困难的表达式

#### 4.点击 “乘方运算”，选择表达式中是否有乘方运算

#### 5.点击 “插入符/米字号”，选择乘方运算的显示格式，“ ^ ” 或 “ ** ”

#### 6.点击 “历史记录”，查看战绩
